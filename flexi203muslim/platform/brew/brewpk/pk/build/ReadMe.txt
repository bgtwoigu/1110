========================================================================
       BREW Simulator Visual Studio Workspace 
========================================================================

This file contains a summary of what you will find in each of the files that
make up the BREW Simulator application.

/////////////////////////////////////////////////////////////////////////////

SimulatorApp.cpp
    This is the main application source file. Please DO NOT modify this file.

/////////////////////////////////////////////////////////////////////////////

Generic OEM Sources 
This folder contains all the OEM source files that are shipped 
in the OEM Porting Kit. These sources can be modified by OEMs to 
simulate the behavior of their device.

/////////////////////////////////////////////////////////////////////////////
 
CoreApps folder 
This folder contains ViewApp implementation. The sources are also 
shipped in the OEM Porting Kit 

/////////////////////////////////////////////////////////////////////////////

Generates BREW_Simulator.exe and BREW_Simulator import library. The import library 
exports various functions that OEMs may require.
