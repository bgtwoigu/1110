Simulator Support DOCUMENTATION

IOBEXSrvSession simulation

IOBEXSrvSession interface is simulated. Following features are supported.

- Retrieve the supported transports
- Set the transport to be used for OBEX transactions
- Create a Header list with OBEX headers
- Authenticate client 
- Provide information required for authentication (such as User ID and password)
- Disconnect from an OBEX client
- Send response to Put, Get, SetPath, Abort and Disconnect operations.

Simulation of REALM information during OBEX Authentication is not supported.

See Also:
IOBEXHeader
IOBEXSrv
IOBEXCli
For more information on IOBEXSrvSession refer the interface documentation.
