Simulator Support DOCUMENTATION

IBTMgr simulation

IBTMgr interface is simulated. 
Following features are supported:

- Retrieve and Set Local Device settings (such as local security level settings, Radio state, Name,Class of device and Role)
- Set Security level for individual services.
- Authorize (Accept or Reject) an incoming Bluetooth connection. 
- Link level connection management

Currently Simulator does not support following API's
IBTMgr_TuneScanPerformance
IBTMgr_SetRoleSwitchPolicy
IBTMgr_SetBlockedDevList



See Also:
IBT
IBTConnection
IBTServiceDiscovery
For more information on IBTMgr interface, refer to interface documentation
