Using QRCT front-end FTM FMRDS commands can be sent by just a click of a mouse.It saves time & effort in copying these commands on QXDM command prompt.
The result can be viewed using QXDM F3 messages.

Help for using QRCT FMRDS front-end:
-----------------------------------
1. Install QRCT 
2. Create a folder "FMRDS" in "C:\Program Files\Qualcomm\QDART\databases\QRCTCustomAPIs\Custom APIs"
3. Copy FMRDS.xml
4. Open QRCT 
5. Click on: custom APIs -> Reload Custom APIs 
5. Click on: custom APIs -> FMRDS -> FMRDS
6. Click on: COM Port -> (whatever port FFA\SURF is connected to eg. Port 9)
7. Start using FTM FMRDS command buttons
8. FTM FMRDS commands with multiple options have combo boxes against them.
9. FMRDS.xml can be easily modified to update the front-end.
